# Normalize Program research

Read every completed research pass and the source inventory. Your task is to
make the passes comparable without deciding the initiative. Do not merge
recommendations into a final answer, write an ExecPlan, or implement code.

Write one `normalized-pass-<slug>.md` or `normalized-track-<N>.md` artifact with:

- `# Normalized Research`
- `## Vocabulary`
- `## Facts`
- `## Agreements`
- `## Conflicts`
- `## Dependencies`
- `## Risks`
- `## Unsupported Claims`
- `## Missing Evidence`
- `## Recommendation`
- `## Source Mapping`

Use one canonical term for each concept while recording aliases. Preserve the
source path for every material normalized claim. Distinguish actual dependency
from preference, fact from inference, and resolvable conflict from a genuine
owner choice. Do not erase minority evidence or manufacture consensus.

Replace only the normalization scaffold for this transition. Return the
artifact path, agreements, unresolved conflicts, missing evidence, and the next
legal transition.
