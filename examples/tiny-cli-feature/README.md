# Tiny CLI feature example

This neutral worked example uses Programmers Loop to add a `greet` command to a
dependency-free Node CLI.

- The completed planning packet lives at
  [`docs/assignments/completed/2026-07-13-tiny-cli-feature/`](../../docs/assignments/completed/2026-07-13-tiny-cli-feature/).
- The tiny implementation lives in [`project/`](project/).
- `bun run test:examples` runs its deterministic acceptance test.

The packet demonstrates an Assignment, Program evidence, convergence,
immutable brief, completed child ExecPlan, and proof without product-specific
vocabulary.
