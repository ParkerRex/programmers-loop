# Skill Index

## Owns

- The inventory and routing for the repository's reusable agent skills.
- The boundary between concise procedure and detailed canonical contracts.

Start with the umbrella [Programmers Loop skill](../../skills/programmers-loop/SKILL.md).
Use [Workshop System](../../skills/workshop-system/SKILL.md) before planning
ambiguous work; [Plan Assignment](../../skills/plan-assignment/SKILL.md),
[Run Program](../../skills/run-program/SKILL.md), and
[Run ExecPlan](../../skills/run-exec-plan/SKILL.md) for artifact lifecycles;
[Run Standup](../../skills/run-standup/SKILL.md) and
[Diagnose Programmers Loop](../../skills/diagnose-programmers-loop/SKILL.md)
for status and health; and
[Maintain Docs Spine](../../skills/maintain-docs-spine/SKILL.md) or
[Verify Programmers Loop](../../skills/verify-programmers-loop/SKILL.md) for
repository maintenance and proof.

## Does Not Own

- Contract details duplicated from canonical docs.
- Runtime implementation or provider-specific flags.
- Installation into a user's global skill directory.
